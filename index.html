About Me

I have learned a lot from my experience with coding. I can honestly say that it has been the greatest experience of my life. And I am looking forward to doing it for the rest of my life.


I am proficient in many different languages and styles of coding. Including HTML, CSS, JavaScript, jQuery, API manipulation, AJAX, Firebase, Node, mySQL, Sequelize, Express, MongoDB and more!


Some of the projects you will find in this portfolio are works in progress. During my Bootcamp, I was given less than two weeks to build these applications from scratch. Since graduation I have been doing freelance coding and working full time. Leaving me little time to debug past projects.


Some of my favorite applications include PriceIt!, and The Broken Moon.


PriceIt! was the first major project I had the pleasure of working on. It uses four different API's to help the user price match products across several retailers such as Amazon, Walmart, and eBay.


The Broken Moon is a dream project of mine. It is a simple game created to showcase MySQL database manipulation using C.R.U.D. operations. I enjoyed this project very much and am proud to say that I created the artwork myself.
